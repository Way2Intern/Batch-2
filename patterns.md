
# Patterns Problem
## 1. Star Pattern
### Problem Statement
Print the following pattern for the given N number of rows.
Pattern for N = 4
```
   *
  ***
 *****
*******
```

### Input Format :
Integer N (Total no. of rows)
### Output Format :
Pattern in N lines
### Sample Input 1 :
```
3
```
### Sample Output 1 :
```
  *
 ***
*****
```

### Sample Input 2 :
```
4
```
### Sample Output 2 :
```
   *
  ***
 *****
*******
```
